# API Backend AstraXR

Backend de la plataforma web y móvil accesible **AstraXR**, diseñada para permitir a personas con discapacidad motriz registrarse, gestionar su perfil, consultar catálogos y realizar compras de manera autónoma.

**Equipo 3:** Carlos Alberto Angeles Alvarez y Esau Yael Gonzalez Leyva.

---

## Framework Elegido
* **Framework:** Express.js [cite: 81]
* **Versión:** 4.19.2 [cite: 81]
* **Entorno:** Node.js [cite: 81]
* **Base de Datos:** PostgreSQL

---

## Requisitos para ejecutar
Para correr este proyecto en entorno local, se necesita tener instalado:
* [Node.js](https://nodejs.org/) (v18 o superior).
* [PostgreSQL](https://www.postgresql.org/) corriendo localmente o en la nube.
* [Git](https://git-scm.com/) para clonar el repositorio.
